Introduction

Python is a versatile, high-level programming language designed for readability and simplicity. It is widely used for web development, data analysis, machine learning, scientific computing, and more.

Python emphasizes code readability with its use of significant indentation. It supports multiple programming paradigms, including procedural, object-oriented, and functional programming.

Its extensive standard library and vibrant ecosystem of third-party libraries make Python one of the most popular programming languages today.

Getting Started

To start using Python, download and install it from the official website. Python is available for Windows, macOS, and Linux.

Once installed, you can execute Python code in an interactive shell (REPL) or save it to a file with a .py extension and run it using the Python interpreter.

Here is an example of a simple Python program:

print("Hello, World!")

Variables and Types

Python variables are dynamically typed, meaning you do not need to declare their type explicitly.

Here are some examples of variable assignment:

x = 10 # Integer y = 3.14 # Float name = "Alice" # String is_active = True # Boolean

Python also supports complex data types such as:

  • list - Ordered, mutable collection
  • tuple - Ordered, immutable collection
  • dict - Key-value pairs
  • set - Unordered collection of unique items

Control Flow

Control flow statements allow you to execute code conditionally or repeatedly.

Python supports the following control flow statements:

  • if and else
  • for loops
  • while loops

Example of an if-else statement:

if x > 0:
  print("Positive")
else:
  print("Non-positive")

Functions

Functions are reusable blocks of code that perform a specific task.

You can define a function using the def keyword:

def greet(name):
  return f"Hello, {name}"
print(greet("Alice"))

Functions can accept parameters and return values, improving code modularity and reusability.

Modules and Packages

Modules are files containing Python code that can be imported and reused in other programs. You can create a module by saving your Python code in a file with a .py extension.

Python also supports packages, which are collections of modules organized in directories. To create a package, include an __init__.py file in the directory.

Example of importing a module:

import math
print(math.sqrt(16))

File Handling

Python provides built-in support for file handling, allowing you to read from and write to files easily.

Example of reading a file:

with open("example.txt", "r") as file:
  content = file.read()
  print(content)

Example of writing to a file:

with open("example.txt", "w") as file:
  file.write("Hello, World!")

Error Handling

Python uses try and except blocks to handle exceptions and prevent runtime errors from crashing your program.

Example:

try:
  result = 10 / 0
except ZeroDivisionError:
  print("Cannot divide by zero")

Advanced Topics

Python offers advanced features such as:

  • Decorators for modifying the behavior of functions
  • Generators for creating iterators
  • Context managers for managing resources
  • Asynchronous programming with async and await

Example of a generator:

def countdown(n):
  while n > 0:
    yield n
    n -= 1
for i in countdown(5):
  print(i)
